The FoodSeg103 dataset is used to train and validate the food image classifier.
The Dataset_loading.ipynb file is used to load the dataset into your local drive.
The image_classifier.ipynb file consists of the training of the Image classifier model.
The final_model.ipynb file gives the code for validation of the model and gives evaluation metrics. 
The final_image_classifier.pth is the saved model.
The nutrition_database.csv is a self created dataset of the calorie values per 100g for 100 different food types.
The UI.py file is a simple python based user interface where food images can be given as an input along with the food quantity and the calorie count will be given.
